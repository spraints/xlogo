History of LOGO
Other help sections: [ How to use XLogo ] [ LOGO Commands ] [ About XLogo ]
  LOGO, a dialect of the programming language Lisp, was primarily developed during the 1970's by MIT researchers. It was designed as a tool for learning, and is still in wide use today. If you would like to read more about LOGO and it's current state, please visit the LOGO Foundation website at http://www.logofoundation.org/
