How to use XLogo
Other help sections: [ LOGO Commands ] [ LOGO History ] [ About XLogo ]
  This help section will show you how to enter in commands and have the turtle draw them, save your commands, load commands from a file, print your drawing or change the speed of the turtle as it draws.

Save your current commands to a file
Click on the first icon at the top, the one with the image of a diskette. This will bring up a small window asking you to select a directory and filename in which to save your commands. Once you do so, select the "OK" button and it will be saved.

Load commands from a file
Click on the second icon at the top, the one with the image of a CD. This will bring up a small window asking you to select a directory and filename from which to load your commands. Once you do so, select the "OK" button the commands will be loaded.

Print your drawing
Click on the third icon at the top, the one with the image of a printer. This will start the printing process.

Changing the speed of the turtle
Before, after, or while the turtle is drawing, you may change the speed at which he draws. You only need to move the slider at the bottom left corner of the XLogo window to either one of the three positions (slow, normal, fast).
