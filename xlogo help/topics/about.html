About XLogo
Other help sections: [ How to use XLogo ] [ LOGO Commands ] [ LOGO History ]
  XLogo is a free and open source application. It was first created by Jeff Skrysak in June 2003. For more information about XLogo, please visit its website at http://xlogo.sourceforge.net/.

XLogo is:
  • Free
  • Open source
  • Designed to be simple to use
  • Written in Cocoa (Objective-C)
  • Available in 3 languages: English, Spanish, French
